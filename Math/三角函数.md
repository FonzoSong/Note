# 三角函数公式大全

## 一、基本定义与符号

### 1. 锐角三角函数（直角三角形定义）
在直角三角形中，设锐角 \(\theta\)：
- **正弦** \(\sin\theta = \dfrac{\text{对边}}{\text{斜边}}\)
- **余弦** \(\cos\theta = \dfrac{\text{邻边}}{\text{斜边}}\)
- **正切** \(\tan\theta = \dfrac{\text{对边}}{\text{邻边}} = \dfrac{\sin\theta}{\cos\theta}\)
- **余切** \(\cot\theta = \dfrac{\text{邻边}}{\text{对边}} = \dfrac{1}{\tan\theta}\)
- **正割** \(\sec\theta = \dfrac{\text{斜边}}{\text{邻边}} = \dfrac{1}{\cos\theta}\)
- **余割** \(\csc\theta = \dfrac{\text{斜边}}{\text{对边}} = \dfrac{1}{\sin\theta}\)

### 2. 任意角三角函数（单位圆定义）
在平面直角坐标系中，以原点为圆心，半径为1的圆称为单位圆。设角 \(\theta\) 的终边与单位圆交于点 \(P(x, y)\)，则：
\[
\sin\theta = y,\quad \cos\theta = x,\quad \tan\theta = \frac{y}{x}\ (x\neq 0)
\]
其余函数由此导出。此定义使三角函数可以推广到任意实数角度。

## 二、核心计算法则

### 1. 倒数关系
\[
\sin\theta \cdot \csc\theta = 1 \qquad
\cos\theta \cdot \sec\theta = 1 \qquad
\tan\theta \cdot \cot\theta = 1
\]

### 2. 平方关系（勾股定理衍生）
\[
\sin^2\theta + \cos^2\theta = 1
\]
\[
1 + \tan^2\theta = \sec^2\theta
\]
\[
1 + \cot^2\theta = \csc^2\theta
\]

### 3. 商数关系
\[
\tan\theta = \frac{\sin\theta}{\cos\theta},\quad \cot\theta = \frac{\cos\theta}{\sin\theta}
\]

---

## 三、诱导法则（奇变偶不变，符号看象限）

将任意角的三角函数转化为锐角三角函数的规则。

**口诀解释：**
- **奇变偶不变**：若角度可写成 \(\frac{\pi}{2} \cdot k \pm \alpha\)（\(k \in \mathbb{Z}\)），当 \(k\) 为奇数时，函数名改变（正弦⇄余弦，正切⇄余切）；当 \(k\) 为偶数时，函数名不变。
- **符号看象限**：将 \(\alpha\) 视为锐角，根据原角所在象限判断原三角函数值的符号，并将该符号放在结果前。

### 常用诱导公式表（\(k\in\mathbb{Z}\)）

| 公式                                        | 结果            | 口诀说明         |
| ------------------------------------------- | --------------- | ---------------- |
| \(\sin(2k\pi + \alpha)\)                    | \(\sin\alpha\)  | 偶不变，一象限正 |
| \(\cos(2k\pi + \alpha)\)                    | \(\cos\alpha\)  | 偶不变，一象限正 |
| \(\tan(2k\pi + \alpha)\)                    | \(\tan\alpha\)  | 偶不变，一象限正 |
| \(\sin(\pi + \alpha)\)                      | \(-\sin\alpha\) | 偶不变，三象限负 |
| \(\cos(\pi + \alpha)\)                      | \(-\cos\alpha\) | 偶不变，三象限负 |
| \(\tan(\pi + \alpha)\)                      | \(\tan\alpha\)  | 偶不变，三象限正 |
| \(\sin(\pi - \alpha)\)                      | \(\sin\alpha\)  | 偶不变，二象限正 |
| \(\cos(\pi - \alpha)\)                      | \(-\cos\alpha\) | 偶不变，二象限负 |
| \(\tan(\pi - \alpha)\)                      | \(-\tan\alpha\) | 偶不变，二象限负 |
| \(\sin\left(\frac{\pi}{2} - \alpha\right)\) | \(\cos\alpha\)  | 奇变，一象限正   |
| \(\cos\left(\frac{\pi}{2} - \alpha\right)\) | \(\sin\alpha\)  | 奇变，一象限正   |
| \(\tan\left(\frac{\pi}{2} - \alpha\right)\) | \(\cot\alpha\)  | 奇变，一象限正   |
| \(\sin\left(\frac{\pi}{2} + \alpha\right)\) | \(\cos\alpha\)  | 奇变，二象限正   |
| \(\cos\left(\frac{\pi}{2} + \alpha\right)\) | \(-\sin\alpha\) | 奇变，二象限负   |
| \(\tan\left(\frac{\pi}{2} + \alpha\right)\) | \(-\cot\alpha\) | 奇变，二象限负   |

## 四、和差角公式
\[
\sin(\alpha \pm \beta) = \sin\alpha \cos\beta \pm \cos\alpha \sin\beta
\]
\[
\cos(\alpha \pm \beta) = \cos\alpha \cos\beta \mp \sin\alpha \sin\beta
\]
\[
\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha \tan\beta}
\]

---

## 五、倍角与半角公式

### 1. 二倍角
\[
\sin 2\theta = 2\sin\theta \cos\theta
\]
\[
\cos 2\theta = \cos^2\theta - \sin^2\theta = 2\cos^2\theta - 1 = 1 - 2\sin^2\theta
\]
\[
\tan 2\theta = \frac{2\tan\theta}{1 - \tan^2\theta}
\]

### 2. 三倍角
\[
\sin 3\theta = 3\sin\theta - 4\sin^3\theta
\]
\[
\cos 3\theta = 4\cos^3\theta - 3\cos\theta
\]

### 3. 半角公式（降幂公式）
\[
\sin^2\frac{\theta}{2} = \frac{1 - \cos\theta}{2},\quad \cos^2\frac{\theta}{2} = \frac{1 + \cos\theta}{2}
\]
\[
\tan\frac{\theta}{2} = \frac{1 - \cos\theta}{\sin\theta} = \frac{\sin\theta}{1 + \cos\theta}
\]

### 4. 万能公式（设 \(t = \tan\frac{\theta}{2}\)）
\[
\sin\theta = \frac{2t}{1+t^2},\quad \cos\theta = \frac{1-t^2}{1+t^2},\quad \tan\theta = \frac{2t}{1-t^2}
\]
常用于积分运算或化简有理分式。

## 六、和差化积与积化和差

### 1. 积化和差
\[
\sin\alpha \cos\beta = \frac{1}{2}[\sin(\alpha+\beta) + \sin(\alpha-\beta)]
\]
\[
\cos\alpha \sin\beta = \frac{1}{2}[\sin(\alpha+\beta) - \sin(\alpha-\beta)]
\]
\[
\cos\alpha \cos\beta = \frac{1}{2}[\cos(\alpha+\beta) + \cos(\alpha-\beta)]
\]
\[
\sin\alpha \sin\beta = -\frac{1}{2}[\cos(\alpha+\beta) - \cos(\alpha-\beta)]
\]

### 2. 和差化积
\[
\sin A + \sin B = 2 \sin\frac{A+B}{2} \cos\frac{A-B}{2}
\]
\[
\sin A - \sin B = 2 \cos\frac{A+B}{2} \sin\frac{A-B}{2}
\]
\[
\cos A + \cos B = 2 \cos\frac{A+B}{2} \cos\frac{A-B}{2}
\]
\[
\cos A - \cos B = -2 \sin\frac{A+B}{2} \sin\frac{A-B}{2}
\]

## 七、解三角形（三角形 \(ABC\)，边 \(a\) 对角 \(A\)）

### 1. 正弦定理
\[
\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R
\]
其中 \(R\) 为外接圆半径。适用于已知两角一边或两边一对角的情形。

### 2. 余弦定理
\[
a^2 = b^2 + c^2 - 2bc \cos A
\]
\[
\cos A = \frac{b^2 + c^2 - a^2}{2bc}
\]
适用于已知三边求角，或已知两边及夹角求第三边。

### 3. 面积公式
\[
S = \frac{1}{2}ab\sin C = \frac{1}{2}bc\sin A = \frac{1}{2}ca\sin B
\]
\[
S = \frac{abc}{4R} \quad (\text{外接圆半径})
\]
\[
S = \sqrt{s(s-a)(s-b)(s-c)} \quad (\text{海伦公式}),\ s=\frac{a+b+c}{2}
\]

## 八、特殊角度的三角函数值

| 角度 \(\theta\)    | \(0^\circ\) |      \(30^\circ\)       |      \(45^\circ\)      |      \(60^\circ\)       | \(90^\circ\) |
| :----------------- | :---------: | :---------------------: | :--------------------: | :---------------------: | :----------: |

## 九、记忆与实用技巧

1. **记忆口诀法**  
   - 定义：Sin 对边比斜边，Cos 邻边比斜边，Tan 对边比邻边。  
   - 诱导公式：“奇变偶不变，符号看象限” 已广为流传。  
   - 平方关系：\(\sin^2\theta + \cos^2\theta = 1\) 是勾股定理的体现，可联想单位圆。

2. **单位圆法**  
   - 单位圆上点的坐标 \((\cos\theta, \sin\theta)\) 直观展示三角函数值随角度的变化，以及各象限符号。  
   - 正切线、余切线等几何意义也有助于理解函数性质。

3. **常见恒等式变形**  
   - \(1 \pm \sin\theta = \left(\sin\frac{\theta}{2} \pm \cos\frac{\theta}{2}\right)^2\)  
   - \(\tan\theta + \cot\theta = \sec\theta \csc\theta\)  
   - \(\sin^4\theta + \cos^4\theta = 1 - 2\sin^2\theta \cos^2\theta = \frac{3 + \cos4\theta}{4}\)

4. **计算器使用法则**  
   - 反三角函数（如 \(\arcsin x\)）返回值通常在主值范围内：  
     \(\arcsin x \in [-\frac{\pi}{2}, \frac{\pi}{2}]\)，\(\arccos x \in [0, \pi]\)，\(\arctan x \in (-\frac{\pi}{2}, \frac{\pi}{2})\)。  
   - 解三角方程时，需根据题目限定的角度范围（如 \([0, 2\pi)\) 或全体实数）加上周期 \(2k\pi\) 或 \(k\pi\)。

5. **辅助角公式**  
   \[
   a\sin x + b\cos x = \sqrt{a^2+b^2}\,\sin(x+\varphi),\quad \varphi = \arctan\frac{b}{a}
   \]
   用于合并同频正弦余弦，简化表达式。

   ## 十、反三角函数
   
   ### 1. 定义与主值区间
   
   三角函数 \(y = \sin x\) 等在其整个定义域上不是一一对应的，因此为了得到反函数，必须限制定义域，使得函数在限制区间上是单调的。这个限制区间称为**主值区间**，得到的反函数称为反三角函数。
   
   | 反函数 | 记法                                                   | 定义域         | 主值区间（值域）                               | 含义                                               |
   | ------ | ------------------------------------------------------ | -------------- | ---------------------------------------------- | -------------------------------------------------- |
   | 反正弦 | \(y = \arcsin x\) 或 \(y = \sin^{-1} x\)               | \([-1, 1]\)    | \(\left[-\frac{\pi}{2}, \frac{\pi}{2}\right]\) | 正弦值为 \(x\) 的角度（在 \([-\pi/2, \pi/2]\) 内） |
   | 反余弦 | \(y = \arccos x\) 或 \(y = \cos^{-1} x\)               | \([-1, 1]\)    | \([0, \pi]\)                                   | 余弦值为 \(x\) 的角度（在 \([0, \pi]\) 内）        |
   | 反正切 | \(y = \arctan x\) 或 \(y = \tan^{-1} x\)               | \(\mathbb{R}\) | \(\left(-\frac{\pi}{2}, \frac{\pi}{2}\right)\) | 正切值为 \(x\) 的角度（在 \((-\pi/2, \pi/2)\) 内） |
   | 反余切 | \(y = \operatorname{arccot} x\) 或 \(y = \cot^{-1} x\) | \(\mathbb{R}\) | \((0, \pi)\)                                   | 余切值为 \(x\) 的角度（在 \((0, \pi)\) 内）        |
   | 反正割 | \(y = \operatorname{arcsec} x\) 或 \(y = \sec^{-1} x\) | \(             | x                                              | \ge 1\)                                            |
   | 反余割 | \(y = \operatorname{arccsc} x\) 或 \(y = \csc^{-1} x\) | \(             | x                                              | \ge 1\)                                            |
   
   **注意**：记号 \(\sin^{-1}x\) 容易与 \((\sin x)^{-1} = \csc x\) 混淆，实际使用中，\(\arcsin x\) 更为清晰。
   
   ### 2. 基本恒等式
   
   - **反函数与原函数复合**：
     \[
     \sin(\arcsin x) = x,\quad x\in[-1,1]; \qquad \arcsin(\sin x) = x,\quad x\in\left[-\frac{\pi}{2},\frac{\pi}{2}\right]
     \]
     \[
     \cos(\arccos x) = x,\quad x\in[-1,1]; \qquad \arccos(\cos x) = x,\quad x\in[0,\pi]
     \]
     \[
     \tan(\arctan x) = x,\quad x\in\mathbb{R}; \qquad \arctan(\tan x) = x,\quad x\in\left(-\frac{\pi}{2},\frac{\pi}{2}\right)
     \]
   
   - **互余关系**：
     \[
     \arcsin x + \arccos x = \frac{\pi}{2},\quad x\in[-1,1]
     \]
     \[
     \arctan x + \operatorname{arccot} x = \frac{\pi}{2},\quad x\in\mathbb{R}
     \]
   
   - **负角关系**：
     \[
     \arcsin(-x) = -\arcsin x,\quad \arccos(-x) = \pi - \arccos x,\quad \arctan(-x) = -\arctan x
     \]
   
   - **倒数关系**（注意定义域）：
     \[
     \operatorname{arccsc} x = \arcsin\frac{1}{x},\quad \operatorname{arcsec} x = \arccos\frac{1}{x},\quad \operatorname{arccot} x = \frac{\pi}{2} - \arctan x
     \]
   
   ### 3. 常用三角恒等式转化
   
   - \(\sin(\arccos x) = \sqrt{1-x^2},\quad \cos(\arcsin x) = \sqrt{1-x^2}\)
   - \(\tan(\arcsin x) = \frac{x}{\sqrt{1-x^2}},\quad \tan(\arccos x) = \frac{\sqrt{1-x^2}}{x}\)
   - \(\sin(\arctan x) = \frac{x}{\sqrt{1+x^2}},\quad \cos(\arctan x) = \frac{1}{\sqrt{1+x^2}}\)
   
   ### 4. 反正切的和差公式
   
   设 \(x, y \in \mathbb{R}\)，且 \(xy \neq 1\)，则
   \[
   \arctan x \pm \arctan y = \arctan\frac{x \pm y}{1 \mp xy} + k\pi
   \]
   其中 \(k\) 需要根据 \(\arctan x \pm \arctan y\) 的实际所在象限确定（通常取 \(k=0\) 或 \(\pm1\)）。常见的简化情形：
   \[
   \arctan x + \arctan y = 
   \begin{cases}
   \arctan\frac{x+y}{1-xy}, & xy < 1,\\
   \arctan\frac{x+y}{1-xy} + \pi, & x>0,\ y>0,\ xy>1,\\
   \arctan\frac{x+y}{1-xy} - \pi, & x<0,\ y<0,\ xy>1.
   \end{cases}
   \]
   
   类似地，有
   \[
   \arctan x - \arctan y = \arctan\frac{x-y}{1+xy}\quad (xy>-1)
   \]
   （同样需注意符号调整）
   
   ### 5. 导数公式
   
   \[
   \frac{d}{dx}\arcsin x = \frac{1}{\sqrt{1-x^2}},\quad |x|<1
   \]
   \[
   \frac{d}{dx}\arccos x = -\frac{1}{\sqrt{1-x^2}},\quad |x|<1
   \]
   \[
   \frac{d}{dx}\arctan x = \frac{1}{1+x^2}
   \]
   \[
   \frac{d}{dx}\operatorname{arccot} x = -\frac{1}{1+x^2}
   \]
   \[
   \frac{d}{dx}\operatorname{arcsec} x = \frac{1}{|x|\sqrt{x^2-1}},\quad |x|>1
   \]
   \[
   \frac{d}{dx}\operatorname{arccsc} x = -\frac{1}{|x|\sqrt{x^2-1}},\quad |x|>1
   \]
   
   ### 6. 积分公式（简单列举）
   
   由导数公式逆向可得：
   \[
   \int \frac{dx}{\sqrt{1-x^2}} = \arcsin x + C
   \]
   \[
   \int \frac{dx}{1+x^2} = \arctan x + C
   \]
   \[
   \int \frac{dx}{x\sqrt{x^2-1}} = \operatorname{arcsec}|x| + C
   \]
   
   ### 7. 应用提示
   
   - 解三角方程时，若求得 \(x = \sin\theta = a\)，则通解为 \(\theta = \arcsin a + 2k\pi\) 或 \(\theta = \pi - \arcsin a + 2k\pi\)（\(k \in \mathbb{Z}\)）。同理，对于余弦、正切也有相应通解公式。
   - 在计算器上求反三角函数时，注意其返回值只在主值区间内，需要结合题目所给条件（如角度象限）调整。

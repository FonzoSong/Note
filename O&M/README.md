### 关于为什么不记录安装手册
当前各个`linux`发行版的发行商都建立了完备的包repo库；
使用包管理器安装各种服务，命令等，都十分的方便；
所以我认为关于安装这方面没什么可讲的；
当然这并不意味着我不会记录由于各种问题，安装服务失败的解决方式；
以及像`k8s`这种需要多个服务组合而成的服务的搭建部署；

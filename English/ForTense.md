### **一、一般现在时（第三人称单数）**

**规则**：
| 动词结尾                 | 变形规则                    | 示例                                  |
| ------------------------ | --------------------------- | ------------------------------------- |
| 大多数动词               | 直接加 **-s**               | work → works, play → plays            |
| **-ch, -sh, -x, -s, -o** | 加 **-es**                  | wash → washes, fix → fixes, go → goes |
| **辅音字母 + y**         | 变 **y** 为 **i** + **-es** | study → studies, carry → carries      |
| **元音字母 + y**         | 直接加 **-s**               | play → plays, buy → buys              |

**例外**：
- **have → has**  
- **do → does**  
- **be → is**  

**注意**：  
1. 以 **-o** 结尾的动词并非全部加 **-es**：  
   - 加 **-es**：go → goes, do → does  
   - 加 **-s**：photo → photos（摄影）, radio → radios

### **二、一般过去时 & 过去分词（规则动词）**
**规则**：
| 动词结尾                         | 变形规则                    | 示例（过去式）                 |
| -------------------------------- | --------------------------- | ------------------------------ |
| 大多数动词                       | 直接加 **-ed**              | work → worked, look → looked   |
| 以 **-e** 结尾                   | 加 **-d**                   | like → liked, dance → danced   |
| **辅音 + 短元音 + 辅音**（重读） | 双写末尾字母 + **-ed**      | stop → stopped, plan → planned |
| **辅音字母 + y**                 | 变 **y** 为 **i** + **-ed** | study → studied, cry → cried   |
| **元音字母 + y**                 | 直接加 **-ed**              | play → played, enjoy → enjoyed |

**不规则动词分类示例**：
| 原型  | 过去式  | 过去分词 | 特点                 |
| ----- | ------- | -------- | -------------------- |
| go    | went    | gone     | 完全变化             |
| eat   | ate     | eaten    | 元音变化             |
| cut   | cut     | cut      | 原型=过去式=过去分词 |
| bring | brought | brought  | 结尾加 `-ought`      |
| write | wrote   | written  | 含 `-en` 结尾        |

---

### **三、现在分词（进行时） & 动名词（-ing形式）**
**规则**：
| 动词结尾                     | 变形规则                      | 示例                                   |
| ---------------------------- | ----------------------------- | -------------------------------------- |
| 大多数动词                   | 直接加 **-ing**               | work → working, play → playing         |
| 以 **-e** 结尾               | 去 **-e** + **-ing**          | write → writing, dance → dancing       |
| **辅音+短元音+辅音**（重读） | 双写末尾字母 + **-ing**       | run → running, swim → swimming         |
| 以 **-ie** 结尾              | 变 **ie** 为 **y** + **-ing** | die → dying, lie → lying               |
| 以 **-c** 结尾               | 加 **-king**                  | picnic → picnicking, panic → panicking |

**例外与注意**：
1. **双写规则限制**：  
   - 非重读音节不双写：`open → opening`（重音在 `o`）  
   - 末尾字母为 `x, w, y` 时不双写：`fix → fixing, show → showing`  
2. **特殊变形**：  
   - `age → ageing`（英式）/ `aging`（美式）  
   - `queue → queueing`（保留 `e`）  

---

### **四、被动语态（be + 过去分词）**
**结构及时态对照**：
| 时态       | 结构                     | 示例                               |
| ---------- | ------------------------ | ---------------------------------- |
| 一般现在时 | am/is/are + 过去分词     | The car **is cleaned**.            |
| 一般过去时 | was/were + 过去分词      | The letter **was sent**.           |
| 现在完成时 | have/has been + 过去分词 | The project **has been finished**. |
| 将来时     | will be + 过去分词       | The results **will be announced**. |

**重点**：  
- **过去分词**必须正确使用规则/不规则形式（如：`break → broken`, not `breaked`）  
- **Be动词**需随主语和时态变化：  
  - `The books were written by him.`（复数过去时）

### **五、深度补充：易错点与例外**
#### 1. **不规则动词的规律性分组**
| 分组                | 示例（原型→过去式→过去分词）       |
| ------------------- | ---------------------------------- |
| **A-A-A型**（不变） | cut→cut→cut, hit→hit→hit           |
| **A-B-B型**         | build→built→built, lend→lent→lent  |
| **A-B-C型**         | sing→sang→sung, drive→drove→driven |
| **特殊变化**        | be→was/were→been, do→did→done      |

#### 2. **拼写混淆详解**
- **双写规则**：  
  - 必须满足：**单音节词** 或 **重音在末音节** + **辅元辅结尾**  
    - 双写：`admit → admitting`（重音在 `mit`）  
    - 不双写：`visit → visiting`（重音在 `vi`）  
- **去 e 规则**：  
  - 加 `-ing` 时去 `e`：`make → making`  
  - 加 `-ed` 时保留 `e`：`hope → hoped`（但 `true → trued` 极少见）  

#### 3. **特殊动词**
| 动词 | 现在分词 | 说明                       |
| ---- | -------- | -------------------------- |
| dye  | dyeing   | 保留 `e` 避免与 `die` 混淆 |
| ski  | skiing   | 直接加 `-ing`（元音结尾）  |

### **六、总结口诀**
>[!tip]
>  **"直加-s, -es 跟五将（-ch/sh/x/s/o）；  
> 辅 y 改 i 再添-es；元 y 照常莫慌张。  
> 过去-ed 有规矩，无声-e 直接-d；  
> 辅元辅且重读，双写尾字再添-ed。  
> -ing 形式更需明，去 e、双写、ie 变 y；  
> 重闭音节双写紧，x/w/y 结尾不双行！"**
